Steem SSE Hints
===================

v3.7.2

If a program doesn't work in Steem SSE, please check the lists below,
maybe the solution is there. 
In some cases it will only work in such setup because of emulation
limitations (eg WS1).


Explanation of symbols
=========================

#: SSE option 'Hacks'

C1: SSE option 'C1: 6850/6301/MIDI/E-Clock'
C2: SSE option 'C2: 68901'

DI: there are bad disk images around, you may want to check at
ataristeven.t15.org for a good version!!

HI: monochrome monitor

STF or STE: select in Machine option 'ST model'

T1XX where X are digits: TOS version

WU or WS and a digit: select in Machine option 'Wake-up state'

WP: write-protect the disk
There are 2 different ways in Steem, depending on whether the disk
is in a ZIP or RAR archive or not.
If it is the case, uncheck Read/Write Archives in the disk manager options.
If not, right click and check Read Only.

ADAT: Disk Manager option 'Accurate Disk Access times'

ACSI: use the SH204.img hard drive image (option 'ACSI hard disk image')

No B: disconnect drive B: in Disk Manager (click on drive B: icon)

digits x digits: select in SSE option 'Display size'

patch: click on the patch icon, select patch, read instructions, apply
